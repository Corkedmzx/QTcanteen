# 数据模型模块

## 概述

数据模型模块定义了系统中使用的核心数据结构，包括菜品、订单和订单项。

## 文件列表

- `dish.h` / `dish.cpp`：菜品数据模型
- `order.h` / `order.cpp`：订单数据模型
- `orderitem.h` / `orderitem.cpp`：订单项数据模型

## 功能说明

### Dish（菜品）

#### 类定义

```cpp
class Dish
{
    int m_id;                    // 菜品ID
    QString m_name;              // 菜品名称
    QString m_category;          // 菜品分类
    double m_price;              // 菜品价格
    QString m_description;       // 菜品描述
    QString m_imagePath;         // 图片路径
};
```

#### 主要方法

```cpp
// 构造函数
Dish()
Dish(const QString &name, const QString &category, double price, 
     const QString &description = "", const QString &imagePath = "")

// Getter/Setter
QString getName() const
void setName(const QString &name)
QString getCategory() const
void setCategory(const QString &category)
double getPrice() const
void setPrice(double price)
QString getDescription() const
void setDescription(const QString &description)
QString getImagePath() const
void setImagePath(const QString &imagePath)
int getId() const
void setId(int id)
```

#### 数据字段说明

- **id**：菜品唯一标识符，自动生成
- **name**：菜品名称，必填
- **category**：菜品分类（如：热菜、凉菜、汤类等）
- **price**：菜品价格，单位：元
- **description**：菜品描述，可选
- **imagePath**：菜品图片路径，可选

### Order（订单）

#### 类定义

```cpp
enum class OrderStatus {
    Pending,      // 待处理
    Processing,  // 处理中
    Completed,   // 已完成
    Cancelled    // 已取消
};

class Order
{
    QString m_orderNo;           // 订单号
    QString m_username;           // 用户名
    QString m_tableNumber;        // 桌号
    int m_customerCount;          // 客户人数
    QVector<OrderItem> m_items;   // 订单项列表
    OrderStatus m_status;         // 订单状态
    QDateTime m_createTime;       // 创建时间
};
```

#### 主要方法

```cpp
// 构造函数
Order()
Order(const QString &orderNo, const QString &username, 
      const QVector<OrderItem> &items, OrderStatus status = OrderStatus::Pending)

// Getter/Setter
QString getOrderNo() const
void setOrderNo(const QString &orderNo)
QString getUsername() const
void setUsername(const QString &username)
QString getTableNumber() const
void setTableNumber(const QString &tableNumber)
int getCustomerCount() const
void setCustomerCount(int count)
QVector<OrderItem> getItems() const
void setItems(const QVector<OrderItem> &items)
OrderStatus getStatus() const
void setStatus(OrderStatus status)
QDateTime getCreateTime() const
void setCreateTime(const QDateTime &time)

// 其他方法
QString getStatusString() const                    // 获取状态字符串
static OrderStatus statusFromString(const QString &statusStr) // 从字符串转换状态
double getTotalAmount() const                      // 计算总金额
```

#### 订单号格式

订单号格式：`MMDD + 3位序号`

示例：
- `0210001`：2月10日第1个订单
- `0305123`：3月5日第123个订单

#### 订单状态

- **Pending（待处理）**：新创建的订单，等待商家处理
- **Processing（处理中）**：商家正在处理订单
- **Completed（已完成）**：订单已完成
- **Cancelled（已取消）**：订单已取消

## 数据关系

### 关系图

```
Order (订单)
  ├── OrderItem (订单项) × N
  │     └── Dish (菜品)
  ├── OrderStatus (状态)
  └── QDateTime (创建时间)
```

### 关系说明

1. **一个订单包含多个订单项**：`Order` 包含 `QVector<OrderItem>`
2. **一个订单项对应一个菜品**：`OrderItem` 包含 `Dish` 对象
3. **订单项有数量属性**：用于计算小计和总金额

## 使用示例

### 创建菜品

```cpp
Dish dish("宫保鸡丁", "热菜", 28.0, "经典川菜", "images/kungpao.jpg");
dish.setId(1);
```

### 创建订单项

```cpp
OrderItem item(dish, 2);  // 2份宫保鸡丁
double total = item.getTotalPrice();  // 56.0
```

### 创建订单

```cpp
QVector<OrderItem> items;
items.append(OrderItem(dish1, 2));
items.append(OrderItem(dish2, 1));

Order order("0210001", "顾客", items, OrderStatus::Pending);
order.setTableNumber("A01");
order.setCustomerCount(2);

double totalAmount = order.getTotalAmount();  // 计算总金额
```

### 状态转换

```cpp
// 字符串转状态
OrderStatus status = Order::statusFromString("待处理");

// 状态转字符串
QString statusStr = order.getStatusString();  // "待处理"
```

## 数据验证

### Dish 验证

- 名称不能为空
- 价格必须大于 0
- ID 必须唯一

### OrderItem 验证

- 数量必须大于 0
- 菜品对象必须有效

### Order 验证

- 订单号不能为空
- 订单项列表不能为空
- 总金额必须大于 0

## 注意事项

1. **数据一致性**：订单项中的菜品信息是创建时的快照，不会随菜品信息更新而改变
2. **价格计算**：总金额基于订单创建时的价格，不受后续价格变动影响
3. **状态管理**：状态转换应该遵循业务逻辑（如：已完成不能改为待处理）
4. **时间戳**：创建时间在订单创建时自动设置，不应手动修改
5. **订单号唯一性**：订单号必须唯一，由系统自动生成
