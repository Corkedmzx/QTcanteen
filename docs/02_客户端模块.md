# 客户端模块

## 概述

客户端模块提供顾客点餐功能，包括菜品浏览、购物车管理、订单提交和结账功能。

## 文件列表

- `clientmainwindow.h` / `clientmainwindow.cpp`：客户端主窗口
- `clientmainwindow.ui`：UI 设计文件

## 功能说明

### 主要功能

1. **菜品浏览**：
   - 按分类显示菜品
   - 显示菜品图片、名称、价格、描述
   - 支持图片加载和显示

2. **购物车管理**：
   - 添加菜品到购物车
   - 修改菜品数量
   - 显示购物车总金额
   - 清空购物车

3. **订单管理**：
   - 提交订单
   - 查看订单详情
   - 结账功能

4. **网络通信**：
   - 自动连接商家端服务器
   - 发送订单数据
   - 接收订单确认
   - 接收菜品更新通知
   - 接收订单完成通知（商家确认订单完成后会收到提示）

## 使用流程

### 菜品浏览流程

1. 窗口启动时自动加载菜品数据
2. 按分类显示菜品（使用 `QGroupBox` 和 `QGridLayout`）
3. 每个菜品显示：
   - 菜品图片（`QLabel`）
   - 菜品名称
   - 菜品价格
   - 菜品描述
   - 数量输入框（`QSpinBox`）
   - "加入购物车"按钮

### 添加菜品到购物车

1. 用户选择菜品并设置数量
2. 点击"加入购物车"按钮
3. 系统检查：
   - 数量是否大于 0
   - 菜品是否已在购物车中
4. 如果已存在，更新数量；否则添加到购物车
5. 更新购物车总金额显示

### 提交订单流程

1. 用户点击"提交订单"按钮
2. 检查购物车是否为空
3. 创建订单对象（`Order`）
4. 生成订单号（格式：MMDD + 3位序号）
5. 通过网络发送订单到商家端
6. 等待订单确认
7. 确认后清空购物车并显示提示

### 结账流程

1. 在订单详情对话框中点击"结账"按钮
2. 发送结账请求到商家端
3. 接收结账成功响应
4. 显示总金额并更新订单状态

### 网络连接流程

1. 窗口创建时自动创建 `NetworkClient` 对象
2. 自动连接到服务器（`localhost:8888`）
3. 连接成功后更新状态栏
4. 连接失败时显示错误信息（但不影响本地功能）
5. 断开连接后不再自动重连（避免在服务器停止时创建不必要的连接）

### 订单完成通知流程

1. 商家端确认订单完成后，服务器会广播订单完成通知
2. 客户端接收到通知后，会显示消息框提示："您的订单 XXX 已完成！"
3. 同时在状态栏显示提示信息

## 数据交互

### 使用的数据管理接口

- `JsonHelper::loadDishes()`：加载菜品数据
- `JsonHelper::saveOrder()`：保存订单数据
- `NetworkClient::sendOrder()`：发送订单
- `NetworkClient::sendCheckoutRequest()`：发送结账请求

### 数据文件

- `dishes.json`：菜品数据
- `orders.json`：订单数据

## UI 组件

### 主要组件

1. **菜品显示区域**：
   - `QScrollArea`：可滚动区域
   - `QGroupBox`：按分类分组
   - `QGridLayout`：网格布局

2. **购物车区域**：
   - `QTableWidget`：购物车表格
   - 显示菜品名称、单价、数量、小计
   - 总金额标签

3. **按钮区域**：
   - "提交订单"按钮
   - "清空购物车"按钮

4. **状态栏**：
   - 显示网络连接状态
   - 显示操作提示信息

## 网络通信

### 连接管理

- 自动连接：窗口创建时自动尝试连接
- 重连机制：连接失败时不影响本地功能
- 状态显示：在状态栏显示连接状态

### 消息类型

1. **发送消息**：
   - `order`：订单数据
   - `checkout`：结账请求

2. **接收消息**：
   - `order_confirm`：订单确认
   - `checkout_success`：结账成功
   - `dish_update`：菜品更新通知

## 图片加载机制

### 图片搜索路径

1. 应用程序目录下的 `images/` 文件夹
2. 当前工作目录下的 `images/` 文件夹
3. 相对路径（相对于可执行文件）

### 图片格式支持

- BMP
- JPG/JPEG
- PNG
- 其他 Qt 支持的图片格式

## 注意事项

1. **网络连接**：网络连接失败不影响本地功能，订单仍会保存到本地文件
2. **图片路径**：确保图片文件在正确的路径下，否则显示占位符
3. **订单号生成**：订单号基于日期和序号，确保唯一性
4. **购物车管理**：购物车数据仅保存在内存中，关闭窗口后清空
