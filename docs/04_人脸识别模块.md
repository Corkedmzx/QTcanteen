# 人脸识别模块

## 概述

人脸识别模块提供人脸检测、识别和录入功能，支持 OpenCV 和 SeetaFace 两种识别方案。

## 文件列表

- `facelogindialog.h` / `facelogindialog.cpp`：人脸识别登录对话框
- `faceenrolldialog.h` / `faceenrolldialog.cpp`：人脸录入对话框
- `opencvfacerecognizer.h` / `opencvfacerecognizer.cpp`：OpenCV 人脸识别实现
- `seetafacefacerecognizer.h` / `seetafacefacerecognizer.cpp`：SeetaFace 人脸识别实现

## 功能说明

### 主要功能

1. **人脸检测**：
   - 实时检测视频流中的人脸
   - 显示检测框和提示信息
   - 支持人脸跟踪（OpenCV）

2. **人脸识别**：
   - 识别已录入的人脸
   - 返回用户名和相似度
   - 支持阈值设置

3. **人脸录入**：
   - 使用摄像头拍照
   - 检测人脸并训练模型
   - 保存人脸数据

### OpenCVFaceRecognizer

#### 主要方法

```cpp
bool initialize()                 // 初始化识别器
bool train(const QString &username, const QImage &faceImage) // 训练
QString recognize(const QImage &image, double &confidence) // 识别
bool detectFace(const QImage &image, QRect &faceRect) // 检测人脸
bool initTracker(const QImage &image, const QRect &faceRect) // 初始化跟踪器
bool updateTracker(const QImage &image, QRect &faceRect) // 更新跟踪器
void resetTracker()              // 重置跟踪器
bool saveModel()                 // 保存模型
bool loadModel()                 // 加载模型
```

#### 识别方案

1. **LBPH 算法**（需要 opencv_contrib）：
   - 使用 `cv::face::LBPHFaceRecognizer`
   - 适合小规模人脸识别

2. **DNN 深度特征**（推荐）：
   - 使用 OpenCV DNN 模块
   - 提取深度特征向量
   - 使用余弦相似度匹配
   - 支持大规模人脸识别

3. **人脸跟踪**（可选）：
   - 使用 `cv::TrackerKCF`（需要 opencv_contrib）
   - 或使用简单位置预测（fallback）

## 使用流程

### 人脸录入流程

1. 在商家端点击"录入人脸"按钮
2. 选择或输入用户名
3. 打开人脸录入对话框
4. 启动摄像头并显示视频流
5. 实时检测人脸（显示绿色检测框）
6. 点击"拍照"按钮
7. 系统检测人脸并显示预览
8. 点击"保存"按钮
9. 保存图像文件到 `faces/` 目录
10. 训练人脸识别模型
11. 保存训练数据

### 人脸识别登录流程

1. 在商家登录对话框中选择"人脸识别登录"
2. 检查是否有人脸数据
3. 打开人脸识别对话框
4. 启动摄像头并显示视频流
5. 实时检测和跟踪人脸
6. 点击"开始识别"按钮
7. 捕获当前帧
8. 调用识别器进行识别
9. 如果识别成功且相似度超过阈值，登录成功
10. 否则显示错误提示

## 技术实现

### 视频显示优化

**问题**：`QVideoWidget` 在 Windows 上使用原生渲染，会遮挡所有子控件

**解决方案**：
- 使用 `QVideoSink` 获取视频帧
- 转换为 `QImage`
- 直接在图像上绘制检测框
- 使用 `QLabel` 显示处理后的图像

### 性能优化

1. **动态调整检测间隔**：
   - 跟踪模式：100ms（更流畅）
   - 检测模式：300ms（降低 CPU 占用）
   - SeetaFace：400ms（检测较慢）

2. **避免帧处理重叠**：
   - 使用 `m_processingFrame` 标志
   - 处理中跳过新帧

3. **优化图像处理**：
   - 仅在需要时进行格式转换
   - 使用快速缩放算法（`Qt::FastTransformation`）
   - 缓存标签尺寸

4. **跟踪优化**：
   - 跟踪比检测快，不需要格式转换
   - 跟踪失败时回退到检测
   - 连续失败 3 次后重新初始化跟踪器

### 人脸检测算法

#### OpenCV

- **Haar Cascade**：使用 `cv::CascadeClassifier`
- 模型文件：`haarcascade_frontalface_alt.xml`
- 支持多尺度检测

#### SeetaFace

- 使用 SeetaFace6Open 的 `FaceDetector`
- 支持更精确的人脸检测
- 支持关键点检测

### 人脸识别算法

#### OpenCV LBPH

- 局部二值模式直方图
- 适合小规模识别
- 需要 opencv_contrib

#### OpenCV DNN

- 使用预训练深度神经网络
- 提取 128 维特征向量
- 使用余弦相似度匹配
- 支持大规模识别

#### SeetaFace

- 使用 SeetaFace6Open 的 `FaceRecognizer`
- 提取 1024 维特征向量
- 使用余弦相似度匹配

## 配置说明

### OpenCV 配置

- 默认路径：`E:/share/opencv452`
- 在 `CMakeLists.txt` 中配置 `OPENCV_ROOT`
- 可选模块：
  - `opencv_contrib`：提供 LBPH 和跟踪功能
  - `opencv_tracking`：提供跟踪功能

### SeetaFace 配置

- 默认路径：`E:/share/SeetaFace`
- 环境变量：`SEETAFACE_ROOT`
- 模型文件路径：`{SEETAFACE_ROOT}/bin/model`
- 必需模型文件：
  - `face_detector.csta` 或 `fd_2_00.dat`
  - `face_landmarker_pts5.csta` 或 `pd_2_00_pts5.dat`
  - `face_recognizer.csta` 或 `fr_2_10.dat`

## 注意事项

1. **摄像头权限**：确保程序有摄像头访问权限
2. **模型文件**：确保模型文件路径正确
3. **识别阈值**：
   - OpenCV：默认 0.6（60% 相似度）
   - SeetaFace：默认 0.55（55% 相似度）
4. **性能**：实时检测会消耗 CPU 资源，建议在性能较好的设备上使用
5. **光照条件**：良好的光照条件有助于提高识别准确率
